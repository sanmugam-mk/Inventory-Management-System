# Inventory Management System

A full-stack web application built using:

- FastAPI (Backend)
- React.js (Frontend)
- PostgreSQL (Database)
- SQLAlchemy

## Features
- Add products
- Update products
- Delete products
- Search functionality
- PostgreSQL persistence

## Setup

### Backend
1. Create virtual environment
2. Install requirements.txt
3. Add .env file
4. Run uvicorn

### Frontend
1. npm install
2. npm start

## To Run the Server

### Backend 
   python -m uvicorn main:app --reload
### Frontend
   npm start
